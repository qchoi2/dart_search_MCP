공시검색 MCP

Claude Desktop에서 DART 공시를 원문 링크와 함께 찾는 확장

설치 준비

1. OpenDART API 인증키 준비
OpenDART에 로그인해 인증키 신청·관리 메뉴에서 40자리 키를 발급받으세요. 키는 다른 사람에게 보내거나 화면 캡처에 포함하지 마세요.
2. 확장 설치
Claude Desktop을 열고 설정 → 확장 → 고급 설정 → 확장 설치를 선택한 뒤 공시검색-MCP-0.3.0.mcpb 파일을 고르세요.
3. 인증키 입력
설치 화면의 OpenDART API 인증키 칸에 키를 입력하고 설치를 완료하세요. 키는 패키지 파일이나 검색 로그에 저장되지 않습니다.

처음 확인할 문장

2025년 1월 1일부터 2025년 12월 31일까지 상계납입 사례를 찾아줘.

각 결과에는 DART 공시 원문 링크가 함께 표시되어야 합니다.

두 가지 검색기능

속도우선 기능은 짧은 시간 안에 우선 확인할 결과를 보여줍니다. 범위가 넓어 모두 확인하지 못하면 심화 검색기능을 안내합니다. 심화 검색은 시작 전에 예상 범위와 시간을 보여주고 사용자의 확인을 받습니다.

질의 유형별 검색 전략

질문의 형태에 따라 확장이 검색 방식과 검증 방식을 자동으로 다르게 적용합니다. 대표적인 유형은 다음과 같습니다.

질의 유형예시검색·검증 전략
정형 개념어 상계납입, 출자전환 미리 정의된 정밀 문구로 본문을 검색하고, 해당 문구가 원문에 있으면 근거로 확정합니다.
문장형·복합 질의 전직·현직 임직원에게 함께 부여된 주식보상 사례 문장을 핵심 개념으로 분해하고 동의어(예: 전직→퇴직·퇴임, 주식보상→주식매수선택권·양도제한조건부주식)로 넓혀 검색합니다. 검증은 모든 핵심 개념이 한 문서 안에 함께 나올 때(동시출현)만 확정하므로, 일부 단어만 걸린 무관한 공시는 제외됩니다.
회사 지정 삼성전자 공시 목록을 보여줘 해당 회사의 공시 목록이나 본문을 대상으로 검색합니다.
관계·순서 분석 정정 전후 비교, 공개매수 후 주식교환, 최종 유효본·철회 여부 정정표·사건 관계를 구조적으로 대조하는 전용 분석으로 처리합니다.

제목만으로 좁히는 조건(예: 제목이 "사업보고서"인 공시만)은 현재 측정·검증이 끝난 뒤 반영할 예정이며, 지금은 본문 핵심어의 동시출현으로 정확도를 확보합니다.

문제 해결

  • 확장이 보이지 않으면 Claude Desktop을 완전히 종료한 뒤 다시 실행하세요.
  • 인증키 오류가 나오면 확장 설정에서 키가 정확한 40자리인지 확인하세요.
  • 제거하려면 설정 → 확장에서 공시검색 MCP를 선택해 제거하세요.
  • 기존 수동 설치가 남아 충돌하면 개발 폴더의 진단 도구로 이전 등록을 확인하세요.

이 도구는 공시 검색·요약을 돕지만 법률 자문을 제공하지 않습니다.

지원 범위: Windows용 Claude Desktop. OpenDART 및 DART의 서비스 상태와 이용 제한에 따라 일부 결과가 부분 제공될 수 있습니다.
