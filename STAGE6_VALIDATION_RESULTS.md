## 제한 라이브 스모크

OpenDART 실제 최종 정정본 3건을 사용했다.

| 유형 | 접수번호 | 최초제출일 | 정정표 | 구조 행 예시 |
|---|---|---|---|---|
| 유상증자 | 20260716000809 | 20260508 | 인식 | 확정예정일: 날짜→추후결정 |
| 합병 | 20260713000345 | 20260521 | 인식 | 합병신주 수 증가 |
| 전환사채 | 20260715000496 | 20260611 | 인식 | 전환가액 증가 |

XML DOCTYPE은 계속 차단한다. 이 때문에 안전 XML 파서가 거부한 합병 문서는 외부 엔터티를 해석하지 않는 HTML fallback으로 텍스트만 처리하고, 표의 행·셀 경계만 보존했다. API 키·Cookie·원문 전체 fixture는 저장하지 않았다.
